How the Pomodoro Timer Works

Work in focused blocks, then take short breaks. Use the left controls to set minutes for Focus, Break, and Long Break, then press Start to begin.

Suggested Setups

  • Classic: Focus 25 min, Break 5 min, Long Break 15 min
  • Deep Work: Focus 50 min, Break 10 min, Long Break 20 min
  • Light Start: Focus 15 min, Break 3 min, Long Break 10 min

Interval Button

The Start Interval button runs an automatic sequence using your Focus/Break/Long Break minutes. Set Interval cycles to the number of focus sessions you want, then press Start Interval to begin immediately.

  • Automatically alternates: Focus → Break → Focus → Break…
  • After the final Break, it starts a Long Break and turns Interval off.
  • While Interval is running, the button changes to Stop Interval (stops Interval and pauses the timer).
